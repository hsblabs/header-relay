Chrome 헤더·요청 도구 비교

API 개발용 헤더/요청 수정 확장 프로그램을 검토 중이라면, 대표적인 세 가지 옵션이 범위와 데이터 처리 방식에서 어떻게 다른지 정리했습니다. ModHeader는 2026년 7월 Google로부터 멀웨어로 판정되어 Edge 부가 기능 스토어에서 삭제되었습니다(출처는 전용 페이지 참고). Requestly는 다기능 오픈소스 HTTP 인터셉터로, 헤더 전용 도구보다 훨씬 넓은 기능을 제공합니다. Header Relay는 셋 중 가장 좁은 범위를 가지며, 단일 역할을 위해 만들어졌습니다.

Header Relay는 ModHeader, Requestly 또는 그 개발자와 제휴 관계가 없습니다. ModHeader 관련 내용은 2026년 7월에 공개된 제3자 보안 조사를, Requestly 관련 내용은 공개 문서와 GitHub 저장소를 기반으로 합니다. 자세한 내용은 연결된 페이지를 확인하세요.
